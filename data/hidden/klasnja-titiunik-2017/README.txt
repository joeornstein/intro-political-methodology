File description


01-run-analysis-bwselect2016.do: 	Stata code to run final analysis with rdrobust version 2016 (for online supplement to supplemental appendix only)

01-run-analysis.do: 			Stata code to run final analysis with rdrobust version 2014

02-prepare-final-figures-LAcountries.R: R file to prepare final Latin America figures

02-prepare-final-figures.R: 		R file to prepare all Brazil figures

02-prepare-final-tables-bwselect2016.R: R file to prepare all final tables using the results in replication-results-bwselect2016.dta	

02-prepare-final-tables.R: 		R file to prepare all final tables using the results in replication-results.dta		

KlasnjaTitiunik-USHouse-data.dta: 	US House election data (1968-2008) used to construct US versus Brazil histogram (in appendix)

functions.do: 				Stata functions to run analyses, called by 01-run-analysis.do and 01-run-analysis-bwselect2016.do

KlasnjaTitiunik-Brazil-data.csv:	Dataset for Brazil analysis, comma-separated-values format 

KlasnjaTitiunik-Brazil-data.dta: 	Dataset for Brazil analysis, Stata 15 format

KlasnjaTitiunik-LAcountries-data.csv: 	Dataset for Latin America analysis, comma-separated-values format 

KlasnjaTitiunik-LAcountries-data.dta: 	Dataset for Latin America analysis, Stata 15 format

replication-results-bwselect2016.dta: 	Results produced by file 01-run-analysis-bwselect2016.do, Stata 15 format

replication-results.dta: 		Results produced by file 01-run-analysis.do, Stata 15 format

README.txt: 				This file

CODEBOOK.txt: 				Codebook explaining the variable names in the main dataset
